Simple Offcanvas for Bootstrap 4!

Offcanvas Plugin is a jQuery plugin that lets you create offcanvas elements, similar to Bootstrap 5's offcanvas. It provides an interactive user experience with a backdrop and automatic closing via backdrop click or the ESC key. Offcanvas is ideal for hiding side navigation or menus without disrupting the main layout.
